Одна из самых больших работ, суть ее - в обучении модели для анализа корректности транзакций. Дополнительно к модели был написан сокет-сервер, веб-сайт для инициализации транзакций (тема - магазин одежды) и приложение для подключения модели и вывода информации по транзакциям в реальном времени и сохранением их в базу данных. Ручных условий для приема транзакций нет, все решения принимаются исключительно моделью.
